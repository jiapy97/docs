# 一、找到适合自己的早晨习惯组合

# 二、具体的梦想
* 越具体和详细，越容易把自己推向正确的方向。
* 把具体的梦想，用便利贴的形式，写在墙上。

# 三、待办事项清单
![](https://jiapeiyang.oss-cn-beijing.aliyuncs.com/mintlify20240609202318.png)

* 每天的待办事项尽量缩短到3个
* 计划要具体、简洁、易于管理

# 四、避开电子产品
* 手机、平板等电子产品放在距离床远的地方
* 早上起床后，不要第一时间看手机

# 五、早上做运动
* 八段锦
* 伸展运动有助于缓解焦虑
